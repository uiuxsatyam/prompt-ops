## 🌟 Overview

**Prompt Ops** is the premium skill vault for autonomous AI agents. Built for top-tier engineering, design, data, and ops teams, it curates rigorous, production-grade logic protocols delivered as raw Markdown (`.md`) files. Whether you're building with Copilot, Cursor, or custom LLM pipelines, these structured architectures act as system contexts to enforce standard practices, global compliance, and flawless output generation.

## ✨ Key Features

- **The Skill Vault**: Explore robust, specialized master prompts spanning Engineering, Design, Testing, and Operations.
- **Copy-Paste Deployment**: Straightforward `.md` files that can instantly be used as system knowledge bases for your AI.
- **Real-World Examples**: View direct examples of Expected Output with interactive previews to understand exactly how the prompt forces compliance.
- **Premium UI / UX**: Deep, dark complementary palettes, glassmorphism, responsive grid layouts, and buttery-smooth animations built via Framer Motion.
- **Security & Privacy First**: Architectures are strictly designed to prevent PII leakage and handle data locally within your LLM environment.

## 🚀 Getting Started

To run the Prompt Ops frontend locally, follow these steps:

### Prerequisites

- Node.js (v16.0 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/uiuxsatyam/prompt-ops.git
   cd prompt-ops
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open in browser**
   Navigate to `http://localhost:5173` to see the live application.
